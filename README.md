# 🧠 BeeCrowd Problem Solutions

Este repositório contém minhas soluções para problemas da plataforma **BeeCrowd (antigo URI Online Judge)**.

O objetivo é registrar minha evolução em **lógica de programação, algoritmos e estruturas de dados**, resolvendo problemas de diferentes níveis de dificuldade.

---

## 📚 Sobre o BeeCrowd

O **BeeCrowd** é uma plataforma amplamente utilizada para prática de programação e preparação para entrevistas técnicas e competições de programação.

Os problemas abordam diversos tópicos da ciência da computação, como:

* Lógica de programação
* Estruturas de dados
* Matemática
* Strings
* Grafos
* Programação dinâmica
* Simulações de algoritmos
* Problemas clássicos da computação

## 📂 Estrutura do repositório

Os problemas estão organizados **por título**, cada um em sua própria pasta.

```text
beecrowd-solutions
│
├── Dama
│   └── Main.java
│
├── pneu
│   └── Main.java
│
├── Array Selection
│   └── main.py
│
└── ...
```

Dentro de cada solução existe um comentário contendo:

* 🆔 **ID do problema no BeeCrowd**
* 📝 **Descrição resumida do problema**
* 💡 **Lógica utilizada para resolver**

Exemplo dentro do código:

```java
/* 1087 - Dama
Descrição:
Determinar o número mínimo de movimentos que uma dama
precisa para sair de uma posição e chegar a outra
em um tabuleiro de xadrez.
*/
```
